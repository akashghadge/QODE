# Mutual Fund Analytics Dashboard

## Live Demo: https://qode-portfolio.netlify.app/

A complete NAV analytics dashboard that loads Excel NAV data, computes financial metrics, plots equity and drawdown graphs, and includes a lightweight JSON-based blog system.

## Overview

This project is a fully client-side financial analytics dashboard built with React.
Features:

Load NAV data from Excel (nav_new.xlsx)

Transform NAV to Equity Curve, Drawdown, Trailing Returns, Monthly Returns

Plot Equity (percent index) + Drawdown (area chart) on a single Y-axis

Export charts as PNG

Local blog system powered by JSON
